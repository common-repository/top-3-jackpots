Major features in Top 3 Lottery Jackpots include:

* Shortcode to insert widget with top 3 jackpots.
* Mobile sticky-footer widget with the first top jackpot (disabled for Home Page by default).
* Widgets have links to website where your visitors can buy lottery tickets (links are bind with your <a href="https://www.24monetize.com/">https://www.24monetize.com/</a> account).

== Installation ==

1. Upload the Top 3 Lottery Jackpots plugin to your site
2. Activate it, and visit your site front-end
3. Add in sidebar standard WordPress "Text" widget and put shortcode [top3jackpots] in the widget body
4. Visit front-end page where the sidebar appeares and click on "Top 3 Lottery Jackpots" tab at admin settings top-bar (appeares there right after the plugin activation) 
5. In appeared settings-box specify User ID (your affiliate id in <a href="https://www.24lottos.com/">https://www.24lottos.com/</a> - to get your 24lottos.com affiliate id, please open an account at <a href="https://www.24monetize.com/">https://www.24monetize.com/</a>)
6. Use "Custom CSS" tab to adjust the widget to your site front-end (Use "Preview" button to quickly see the changes). Or use standard your theme custom CSS feature for this if available.
7. Turn switch "Is Public" to "Yes" and click on "Save" button.
